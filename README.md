# Stellar burgers

Проект в рамках курса «Мидл фронтенд-разработчик и модуль по React» от Яндекс Практикум

Установка зависимостей

```bash
npm install
```

Запуск dev-сервера

```bash
npm run dev
```

Сборка проекта

```bash
npm run build
```

Запуск тестов

```bash
npm run test:unit
npm run test:e2e
```

## Технологии

Основные технологии
- React
- Redux Toolkit
- React Router
- React DnD
- TypeScript
- Vite
- ESLint и Prettier
- Vitest
- Playwright
